Database Management System

DBMS

It is a software used to create, store and manipulate databases.

DATABASE

It is the information stored in an organised way so that it can be accessed and manipulated easily. We can create database through software or as hard copies with pen and paper in the form of tables, records etc.

Types of databases

  • Relational databse
  • Non-relational database

Relational Database

In Relational Database, data is stored in the form of tables. It is a collection of data with predefined relationships. It is most popular form of data and widely used.
Example: SQL

Non-relational Database

A non-relational database is a database that does not use the tabular schema of rows and columns found in most traditional database systems. It generally stores data in form of objects.
Example: No SQL

Structured Query Language (SQL)

It is used to store and manage relational databases. It is the old and mature method. It includes table structure i.e. data is stored in the form of rows and columns. It becomes bulky in size. It is less flexible to changes. It requires schema i.e. what rows and columns will be required in future to create database and that can not be altered.
Example: MySQL

Not Only Structured Query Language

It stores data in the form of objects. It is new and shiny method of storing data. It have document structure and is more flexible to change. We cant deal with complex relationships with it but it is easily scalable.
Example: mongoDB
